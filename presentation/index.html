Overview

Parallel Assignment

Vorbild Python 

							a, b, c = 1, 2, 3
							print(a, b, c) // "1 2 3"
IML 

								// with var a, b, c: int64;
								a init, b init := 1, 2;
								// a = 1, b = 2
								a, b, c init := b, a, a + b;
								// a = 2, b = 1, c = 3

Anwendungsfälle


  • Kompakte Mehrzuweisung
  • Swapping von Variablen

Ternäres Konditional


							// with var a, b, c: int64;
							a init := 12; b init := 20;
							// c mit -1, 0, 1 füllen aufgrund von a <=> b?
							
							c init := a > b ? 1 : a < b ? -1 : 0;
							// c = -1

							// Expressions:

Einschränkungen

Parallel Assignment

Kontexteinschränkungen

  • init pro Variable

							// with var a, b, c: int64;
							b init := 1;
							// b = 1
							a init, b, c init := 1, 2, 3;
							// a = 1, b = 2, c = 3

Typeinschränkungen

  • Einzeln uneingeschränkt (bool, int64)
  • Gleiche Zahl und Reihenfolge 
    
								// var b1,b2: bool; var i1,i2: int64;
								// all 4 initialized
								b1, i1, i2, b2 := true, 1, 2, false;

Ternäres Konditional

Kontexteinschränkungen

lExpr
  • Liefert Adresse
  • Momentan: Variable
rExpr
  • Liefert Wert
  • 17
  • a
  • x + 2
  • f(17)
  • x == 10 ? 1 : 0

Kontexteinschränkungen


							proc p1(in    copy   a:int64, in    ref  b:int64,
							        inout copy   c:int64, inout ref  d:int64,
							        out   copy   e:int64, out   ref  f:int64)
							do
							    // Implementation omitted.
							endproc;

							lExpr := rExpr;
							p1(rExpr, lExpr,
							   lExpr, lExpr,
							   lExpr, lExpr);

Typeinschränkungen

e1 ? e2 : e3
  • type(e1) == bool
  • type(e2) == type(e3)
  • type(e1 ? e2 : e3) == type(e2)
